1. HTML - HyperText Markup Language

1.1 Was ist HTML?

HTML ist eine Auszeichnungssprache (Markup Language) zur Strukturierung von Webinhalten. Sie wurde von Tim Berners-Lee am CERN entwickelt (1989).

HTML definiert die STRUKTUR einer Webseite, nicht das Design. Das Design wird mit CSS gemacht!

1.2 Grundstruktur eines HTML-Dokuments

<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>Seitentitel</title>
    <link href="style.css" rel="stylesheet">
</head>
<body>
    <!-- Hier kommt der sichtbare Inhalt -->
    <h1>Meine erste Website</h1>
    <p>Ein Absatz mit Text.</p>
</body>
</html>

2. CSS - Cascading Style Sheets

2.1 Was ist CSS?

CSS definiert das visuelle Design von HTML-Elementen. "Cascading" bedeutet, dass Styles auf untergeordnete Elemente vererbt werden.

Trennung von Struktur (HTML) und Design (CSS) macht den Code ubersichtlicher und wartbarer!

2.2 CSS einbinden - 3 Methoden

1. Inline-Style (direkt im Element)

<h1 style="color: red; font-size: 24px;">Roter Text</h1>

2. Internal Stylesheet (im <head>)

<head>
    <style>
        h1 {
            color: blue;
            font-size: 24px;
        }
    </style>
</head>

3. External Stylesheet (separate Datei - EMPFOHLEN!)

<!-- In der HTML-Datei: -->
<head>
    <link href="style.css" rel="stylesheet">
</head>

/* In style.css: */
h1 {
    color: green;
    font-size: 24px;
}

2.3 CSS Selektoren

Selektor Syntax Beispiel Prioritaet
Universal * * { margin: 0; } Niedrigste
Element/Tag element p { color: red; }
Klasse .klassenname .highlight { background: yellow; }
ID #idname #header { height: 100px; }
Attribut [attr="wert"] input[type="text"] { ... }
Pseudoklasse :zustand a:hover { color: red; }
Inline-Style style="..." style="color: red;" Hochste

Selektoren kombinieren

/* Alle p in einem div */
div p { color: blue; }

/* Element mit Klasse */
p.wichtig { font-weight: bold; }

/* Mehrere Selektoren */
h1, h2, h3 { color: navy; }

/* Direktes Kind */
div > p { margin: 10px; }

/* Nachbar-Element */
h1 + p { font-size: 1.2em; }

2.4 Das Box-Modell

/*
+----------------------------------+
|           MARGIN                 |
|  +----------------------------+  |
|  |        BORDER              |  |
|  |  +----------------------+  |  |
|  |  |      PADDING         |  |  |
|  |  |  +----------------+  |  |  |
|  |  |  |    CONTENT     |  |  |  |
|  |  |  +----------------+  |  |  |
|  |  +----------------------+  |  |
|  +----------------------------+  |
+----------------------------------+
*/

.box {
    width: 200px;           /* Breite des Inhalts */
    height: 100px;          /* Hoehe des Inhalts */
    padding: 20px;          /* Innenabstand */
    border: 2px solid black;/* Rahmen */
    margin: 10px;           /* Aussenabstand */

    /* WICHTIG: border-box schliesst padding+border in width ein */
    box-sizing: border-box;
}

2.5 Wichtige CSS-Eigenschaften

Farben

/* Verschiedene Farbformate */
color: red;                    /* Name */

color: rgb(255, 0, 0);        /* RGB */
color: rgba(255, 0, 0, 0.5);  /* RGB mit Transparenz */

background-color: lightblue;
background: linear-gradient(to right, red, blue);

Text & Schrift

font-family: Arial, sans-serif;
font-size: 16px;           /* oder: 1em, 1rem, 120% */
font-weight: bold;         /* oder: 100-900 */
font-style: italic;
text-align: center;        /* left, right, justify */
text-decoration: underline;/* none, line-through */

Display & Positionierung

display: block;        /* Nimmt volle Breite */
display: inline;       /* Nur so breit wie Inhalt */
display: inline-block; /* Inline mit Block-Eigenschaften */
display: none;         /* Versteckt */
display: flex;         /* Flexbox Layout */

position: static;      /* Normal (Standard) */
position: relative;    /* Relativ zu Normal-Position */
position: absolute;    /* Relativ zu positioniertem Parent */
position: fixed;       /* Relativ zum Viewport */
position: sticky;      /* Hybrid aus relative+fixed */

Flexbox (wichtig!)

.container {
    display: flex;
    flex-direction: row;       /* oder: column */
    justify-content: center;   /* Hauptachse */
    align-items: center;       /* Querachse */
    gap: 10px;                 /* Abstand zwischen Items */
    flex-wrap: wrap;           /* Umbruch erlauben */
}

.item {
    flex: 1;                   /* Gleichmaessig verteilen */
}

2.6 Responsive Design mit Media Queries

/* Styles fur kleine Bildschirme (max 600px) */
@media (max-width: 600px) {
    body {
        font-size: 14px;
    }

    .container {
        flex-direction: column;
    }

    nav {
        display: none;  /* Navigation verstecken */
    }
}

/* Styles fur grosse Bildschirme */
@media (min-width: 1200px) {
    .container {
        max-width: 1100px;
    }
}

3. JavaScript - Clientseitige Programmierung

3.1 Was ist JavaScript?

JavaScript macht Webseiten interaktiv. Es lauft im Browser (clientseitig) und kann HTML/CSS dynamisch verandern.

3.2 JavaScript einbinden

<!-- Inline (im HTML) -->
<button onclick="alert('Hallo!')">Klick mich</button>

<!-- Internal Script (im HTML) -->
<script>
    function sagHallo() {
        alert('Hallo!');
    }
</script>

<!-- External Script (EMPFOHLEN) -->
<script src="script.js"></script>

3.3 Variablen & Datentypen

// Variablen deklarieren
let name = "Max";           // String (Text)
let alter = 25;             // Number (Zahl)
let istStudent = true;      // Boolean (true/false)
let liste = [1, 2, 3];      // Array (Liste)
let person = {              // Object
    name: "Max",
    alter: 25
};

// const = Konstante (nicht anderbar)
const PI = 3.14159;

// var = alte Schreibweise (nicht empfohlen)
var altesFormat = "vermeiden";

3.4 Operatoren

// Arithmetisch
let summe = 5 + 3;      // 8
let differenz = 5 - 3;  // 2
let produkt = 5 * 3;    // 15
let quotient = 6 / 2;   // 3
let rest = 7 % 3;       // 1 (Modulo)

// Vergleich
5 == "5"    // true  (Wert-Vergleich)
5 === "5"   // false (Typ+Wert-Vergleich) - EMPFOHLEN!
5 !== 3     // true  (ungleich)
5 > 3       // true
5 >= 5      // true

// Logisch
true && false  // false (UND)
true || false  // true  (ODER)
!true          // false (NICHT)

3.5 Kontrollstrukturen

// If-Else
if (alter >= 18) {
    console.log("Erwachsen");
} else if (alter >= 13) {
    console.log("Teenager");
} else {
    console.log("Kind");
}

// Switch
switch (note) {
    case 1:
        console.log("Sehr gut");
        break;
    case 2:
        console.log("Gut");
        break;
    default:
        console.log("Andere Note");
}

// For-Schleife
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4
}

// For-of (durch Array)
let farben = ["rot", "gruen", "blau"];
for (let farbe of farben) {
    console.log(farbe);
}

// While-Schleife
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

3.6 Funktionen

// Funktion definieren
function begruessen(name) {
    return "Hallo, " + name + "!";
}

// Funktion aufrufen
let nachricht = begruessen("Max");  // "Hallo, Max!"

// Arrow Function (moderne Schreibweise)
const addieren = (a, b) => a + b;
let ergebnis = addieren(3, 4);  // 7

// Funktion mit mehreren Zeilen
const berechne = (a, b) => {
    let summe = a + b;
    let produkt = a * b;
    return { summe, produkt };
};

3.8 Events (Ereignisse)

// Methode 1: Im HTML
<button onclick="meineFunktion()">Klick</button>

// Methode 2: addEventListener (EMPFOHLEN)
let button = document.getElementById("meinButton");
button.addEventListener("click", function() {
    alert("Geklickt!");
});

// Wichtige Events:
// click       - Mausklick
// dblclick    - Doppelklick
// mouseover   - Maus uber Element
// mouseout    - Maus verlasst Element
// keydown     - Taste gedruckt
// keyup       - Taste losgelassen
// change      - Wert geandert (Input, Select)
// submit      - Formular abgeschickt
// load        - Seite geladen

// Event-Objekt nutzen
document.addEventListener("keydown", function(event) {
    console.log("Taste: " + event.key);
    console.log("Keycode: " + event.keyCode);
});

Aufgabe 7: Rechner mit +/-

Losung anzeigen 
<label>Zahl 1: <input type="number" id="z1"></label>
<label>Zahl 2: <input type="number" id="z2"></label>
<select id="op">
    <option value="+">+</option>
    <option value="-">-</option>
</select>
<button onclick="rechnen()">Rechnen!</button>
<p>Ergebnis: <input type="text" id="ergebnis" readonly></p>

<script>
function rechnen() {
    let z1 = parseFloat(document.getElementById('z1').value);
    let z2 = parseFloat(document.getElementById('z2').value);
    let op = document.getElementById('op').value;
    let result;

    if (op === '+') {
        result = z1 + z2;
    } else {
        result = z1 - z2;
    }

    document.getElementById('ergebnis').value = result;
}
</script>
